INSTALL

can-eth-utils
============

Requirements:
-------------

+ gcc
+ make
+ pkg-config
+ libnl-3
+ libnl-3-dev
+ libnl-genl-3
+ libnl-genl-3-dev

Installation
------------

	make
	make install


Usage
-----

look for [manual](https://github.com/can-eth-gw/cegwctl/blob/master/man/cegwctl.8.md)

	man cegwctl

how to use the command.
